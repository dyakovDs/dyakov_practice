# Мониторинг экосистемы через IT-решения — прототип распознавания вредных насекомых

**Краткое описание.**  
Этот репозиторий содержит минимальный прототип решения для автоматической классификации опасных сельскохозяйственных насекомых по фотографиям. В качестве источника предобученной модели используется модель с Hugging Face (`dima806/farm_insects_image_detection`). Проект демонстрирует end-to-end поток: загрузка изображения → inference → массовая оценка на датасете → метрики и выводы.

## Проблема (контекст курса «Мониторинг экосистемы через IT-решения»)
Окружающая среда и агроэкосистемы подвергаются постоянным изменениям (климат, антропогенное воздействие). Экологам и фермерам нужны оперативные и надёжные данные для мониторинга и принятия решений. Ручные методы учёта видов — долгие и подвержены ошибкам. Автоматизация распознавания видов по фото позволяет собирать массовые наблюдения, раннее обнаружение вспышек вредителей и оперативную реакцию.

---

## Решение (что реализовано)
- Используем предобученную модель `dima806/farm_insects_image_detection` через `transformers` pipeline.
- `main.py` — минимальный скрипт для запуска inference по одному изображению.
- `eval_on_dataset.py` — скрипт для пакетного прогона по папке `farm_insects/`, сравнения топ-1 предсказания с именем папки (истинной меткой) и вычисления accuracy; опционально — сохраниение результатов в CSV и построение confusion matrix в ноутбуке.
- `main.ipynb` — ноутбук с инструкциями по загрузке датасета (Kaggle), примерами запуска и визуализацией результатов.

---

## О датасете
**Dangerous Farm Insects Image Dataset** — коллекция изображений 15 видов насекомых, распространённых в сельском хозяйстве. Применение:
- идентификация вредителей и управление ними;
- защита урожая и повышение урожайности;
- учебные и исследовательские задачи;
- тренировка и оценка моделей компьютерного зрения.

Ключевые слова: `farm insects, pest management, crop protection, agriculture, entomology, computer vision, machine learning`.
