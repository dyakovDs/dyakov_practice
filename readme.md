# Мониторинг экосистемы через IT-решения — прототип распознавания вредных насекомых

**Краткое описание.**  
Этот репозиторий содержит MVP решения для автоматической классификации опасных сельскохозяйственных насекомых по фотографиям. В качестве источника предобученной модели используется модель с Hugging Face (`dima806/farm_insects_image_detection`). Проект демонстрирует end-to-end поток: загрузка изображения → inference → массовая оценка на датасете → метрики и выводы.


**Структура проекта**

    spider-classifier/                  # корень проекта
      ├── README.md                     # этот файл
      ├── requirements.txt              # зависимости
      ├── main.ipynb                    # ноутбук с пошаговой демонстрацией  
      └── farm_insects/                 # датасет (15 подпапок — по видам)  
        ├── Armyworms/
        ├── Fall Armyworms/
        └── …


**Проблема (контекст курса «Мониторинг экосистемы через IT-решения»)**

Окружающая среда и агроэкосистемы подвергаются постоянным изменениям (климат, антропогенное воздействие). Экологам и фермерам нужны оперативные, надёжные данные и инструменты работы с ними для мониторинга и принятия решений. Ручные методы учёта видов насекомых в этой области — долгие и подвержены ошибкам. Автоматизация распознавания видов по фото позволяет собирать массовые наблюдения, осуществлять раннее обнаружение вспышек вредителей и оперативно принимать соответствующие меры.


**Решение (что реализовано)**
- Найдена предобученную модель `dima806/farm_insects_image_detection` для классификации типичных вредителей сельского хозяйства. Модель запускается через `transformers` pipeline. Модель найдена на платформе huggingface.
- `main.ipynb` — ноутбук с импортом зависимостей, иннициализацией модели и демонстрацией её работы.


**О датасете**

**Dangerous Farm Insects Image Dataset** — коллекция изображений 15 видов насекомых, распространённых в сельском хозяйстве. Применение:
- идентификация вредителей;
- защита урожая и повышение урожайности;
- учебные и исследовательские задачи;
- тренировка и оценка моделей компьютерного зрения.

**О модели**
Модель на базе VIT (visual transformer) обучена на датасете, содержащем 15 видов насекомых - то есть решает задачу классификации. Модель разработана на PyTorch. На полном датасете accuracy достигает 91%, что говорит о высокой точности в решении стоящей задачи. 
Страница на huggingface:
https://huggingface.co/dima806/farm_insects_image_detection

**Инструкция по установке**
1. Клонируйте репозиторий
```
git clone https://github.com/dyakovDs/dyakov_practice.git
cd dyakov_practice
```
3. Создайте виртуальное окружение
macOS / Linux:
```
python3 -m venv .venv
source .venv/bin/activate
```
Windows (PowerShell):
```
python -m venv .venv
.\.venv\Scripts\activate
```
5. Установите зависимости
```
pip install -r requirements.txt
```
7. Подготовьте датасет
В корне проекта должна быть папка:
```
farm_insects/
    ├── Aphids/
    ├── Armyworms/
    ├── Fruit Flies/
    └── ...
```
Если её нет — скачайте датасет или поместите свои изображения.

9. Используя файл main.ipynb запускайте поочередно ячейки для воспроизведения результата.

**Потенциал внедрения**
Такую модель можно применять на фермах для автоматического обнаружения вредителей по фотографиям растений, тем самым ускоряя выявление проблем и снижая потери урожая. Она может интегрироваться в мобильное приложение, чтобы агрономы и фермеры могли сразу определить насекомое прямо "в полях". Также классификатор можно встроить в дроны или автоматизированные камеры, которые регулярно сканируют посадки. Это помогает принимать решения о точечном применении пестицидов, сокращая расходы и повышая экологичность. В итоге технология повышает эффективность сельского хозяйства и уменьшает риски заражения культур.
